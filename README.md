# Soweto Muslim Shura Council — Website Redesign

> **Advocacy · Development · Unity**

A full static website redesign for the [Soweto Muslim Shura Council](https://www.sowetoshura.org.za), built with pure HTML, CSS, and vanilla JavaScript. No frameworks, no dependencies — just clean, fast, and fully responsive code.

## ✨ Features

- **#54B435** brand green throughout — consistent with organisational identity
- **Responsive** — fully mobile-friendly, tested down to 320px
- **Hadith sidebar** — random hadith generator on the homepage hero with Arabic text, translation, and source
- **Animated stats** — count-up animation on scroll for all key figures
- **Scroll reveal** — elements animate in as you scroll
- **Masonry gallery** — photo grid with lightbox (keyboard navigable)
- **Business directory** — live search + category filter chips
- **Contact form** — styled form with subject dropdown and success state
- **Fixed nav** — blurred dark navbar with mobile hamburger menu

## 🚀 How to Deploy on GitHub Pages

1. Create a new GitHub repository named `soweto-shura-site`
2. Upload all files maintaining the folder structure above
3. Go to **Settings → Pages**
4. Set source to `main` branch, `/ (root)` folder
5. Click **Save** — your site will be live in ~60 seconds
